# fabric/

Dossier réservé à l'intégration Git native de Microsoft Fabric (Workspace Settings → Git integration), distincte de `rayfin/` (qui gère le déploiement de l'app via le CLI Rayfin).

Une fois le workspace connecté à ce dossier depuis le portail Fabric, les items du workspace (rapports, lakehouses, notebooks, modèles sémantiques...) seront synchronisés ici automatiquement.
