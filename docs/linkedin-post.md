⚡ J'ai testé Rayfin, le SDK/CLI open-source de Microsoft pour construire des applications d'entreprise sur Microsoft Fabric. Couplé à Claude Code, tout se pilote en discutant avec l'IA, sans ouvrir un IDE.

Concrètement, ça permet de :
✅ Récupérer de la donnée externe
✅ La transformer selon une logique métier
✅ L'enregistrer et la modifier dans une base SQL, aussi simple qu'un tableur Excel
✅ Suivre des KPI et un dashboard

Le tout connecté et fonctionnel dans une seule application, sans développement classique.

Pour les profils non-développeurs, ça change la donne : on peut aujourd'hui construire une vraie application métier de bout en bout — frontend, backend, base de données, dashboard — sans être ingénieur logiciel.

🔍 Prochaine étape : creuser si le dashboard peut rivaliser avec un rapport Power BI classique.
